# Ziying Li — Personal Portfolio

A minimal, clean personal portfolio website for Ziying Li (Lisa), a designer pursuing an HCI master's at UW.

## Structure

```
├── index.html        # Home page (hero, works grid, awards, contact)
├── kwai.html         # Kwai App project page
├── kakawaii.html     # Kaka&Waii project page
├── nikki.html        # Infinity Nikki project page
├── styles.css        # All styles
├── main.js           # Scroll reveal + nav interactions
└── assets/           # Place project cover images here (kwai-cover.jpg, kaka-cover.jpg, nikki-cover.jpg)
```

## Adding Project Images

Drop cover images into the `assets/` folder:
- `assets/kwai-cover.jpg` — Kwai App cover
- `assets/kaka-cover.jpg` — Kaka&Waii cover
- `assets/nikki-cover.jpg` — Infinity Nikki cover

Images will automatically appear over the gradient placeholders.
